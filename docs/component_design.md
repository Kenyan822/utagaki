# コンポーネント設計 (Component Design)

## デザインシステム・共通コンポーネント (Atoms/Molecules)

### Layout & Base

- **`BaseLayout`**:
  - 全画面共通のレイアウト。
  - 背景色（生成色）、フォント（明朝体/筆文字）の設定。
  - モバイルファーストなコンテナ設定。
- **`Navigation`**:
  - 下部タブバー（モバイル用）。和風アイコンを使用。

### Typography & Display

- **`VerticalText`**:
  - `writing-mode: vertical-rl` を適用したテキストコンテナ。
  - 句の表示に使用。
- **`TanzakuCard` (短冊カード)**:
  - 上の句を表示するための縦長のカードコンポーネント。
  - 背景色や柄のバリエーション（色とりどりの和紙）。
- **`MaskIcon` (面アイコン)**:
  - ユーザーアバター表示用。
  - `variant`: 家紋, 狐面, 般若, etc.

### Inputs & Actions

- **`BrushInput` (筆文字入力)**:
  - テキストエリア。入力した文字が筆文字風フォントで表示される。
  - 縦書き入力対応（CSS で制御）。
- **`MusubiButton` (結びボタン)**:
  - マッチング/いいね用のアクションボタン。
  - 朱色を基調とし、水引や結び目のアイコンを添える。

## 機能別コンポーネント (Organisms)

### 1. マッチング・タイムライン (River)

- **`RiverStream`**:
  - 男性ホーム画面。背景に川のアニメーション。
  - `TanzakuCard`が横/斜めに流れる動きを制御するコンテナ。
- **`VerseDetailModal`**:
  - 短冊をタップした際に表示される詳細＆返信フォーム。

### 2. 投稿・返信

- **`VerseComposer`**:
  - 上の句投稿用フォーム。
  - 文字数カウント、プレビュー機能。
- **`ReplyComposer`**:
  - 下の句返信用フォーム。

### 3. マッチング結果・チャット

- **`MatchRevealOverlay`**:
  - マッチング成立時の「御簾が上がる/扇子が開く」全画面アニメーション。
- **`ChatRoom`**:
  - メッセージリストと送信フォーム。
  - 吹き出しデザインも和紙風や巻物風にアレンジ。

### 4. オンボーディング

- **`AvatarGenerator`**:
  - 「面」を選択・生成する UI。
- **`PhotoUploader`**:
  - 顔写真アップロード（ぼかしプレビュー等の処理含む）。
