# デザイン・UI/UX 要件 - Utagaki

## 1. 世界観・デザインコンセプト

**テーマ**: **和風モダン (Wa-fu Modern)**

- 日本の伝統的な美意識と現代的な Web UI の融合。
- 縦書き（Vertical Writing）を効果的に使用しつつ、ユーザビリティを損なわないバランスを目指す。

**配色 (Japanese Traditional Colors)**:

- **ベース**: 生成色（きなりいろ #fcfaf2） - 和紙のような風合い
- **メイン**: 藍色（あいいろ #165e83） - 落ち着き、信頼
- **サブ**: 檜皮色（ひわだいろ #965042） - 伝統的、木材
- **アクセント**: 朱色（しゅいろ #eb6101） - 鳥居、結び、強調
- **若草色** (#c3d825) - 新しさ、通知など

**フォント**:

- システム標準ゴシックは避ける。
- **明朝体**: `Noto Serif JP` または `Shippori Mincho` (本文、システム UI)
- **筆文字**: `Yuji Syuku` 等 (投稿された句、強調部分)

## 3. 主要画面 UI イメージ

### A. プロフィール画像 (マッチング前)

- **「面 (おもて)」**:
  - 顔写真の代わりに表示されるアバター。
  - 登録時に選択: 鳥獣戯画風の動物、妖怪、家紋、能面など。
  - 「顔は見えないが、雰囲気や性格（好みのアイコン）は伝わる」状態。

### B. ホーム画面 (男性側: 詠むを探す)

- **UI コンセプト: 「川」**:
  - 画面上を「川」が流れており、そこを「短冊」「紅葉」「笹舟」に乗った上の句がどんぶらこと流れてくる。
  - 縦スクロールではなく、ゆっくりとした自動スクロールまたは横スクロール等を検討。
  - 気になったものをタップして拾い上げ、詳細を確認して「返歌」する。

### C. 投稿・返信画面

- **「半紙」**:
  - 入力フォームは和紙のテクスチャ。
  - 入力された文字は縦書き（`writing-mode: vertical-rl`）で表示。
  - 筆文字フォントを適用。
- **補助機能**:
  - 字余り・字足らず判定。
  - 季語サジェスト（余裕があれば）。

### D. マッチング成立演出 (The Wow Factor)

- **「御簾（みす）が上がる」 / 「扇子が開く」**:
  - 女性が「結び」ボタンを押した瞬間のアニメーション。
  - 御簾が上がり、それまで隠されていた相手の「顔写真」と「詳細プロフィール（身上書）」が解禁される。
  - ドキドキ感を最大化する演出。
