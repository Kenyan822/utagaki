# サイトマップ / 画面遷移図

## 共通

- **LP / トップページ**: アプリ紹介、ログイン/新規登録導線
- **ログイン/登録画面**:
  - ソーシャルログイン (LINE, Google)
  - メールアドレス登録
- **オンボーディング (初回登録)**:
  - プロフィール入力（ニックネーム、年齢、居住地）
  - 「面（アイコン）」選択
  - 「顔写真」アップロード（※マッチングまで非公開）
  - 身分証アップロード（※後回し可とするか要検討、機能制限あり）

## 女性ユーザー

1. **ホーム (詠む)**
   - 上の句投稿フォーム（筆文字・縦書き）
   - 自分の投稿履歴（有効期限タイマー表示）
2. **待合処 (回答一覧)**
   - 自分の上の句ごとの「返歌」リスト
   - 返歌詳細・プロフィール（アイコンのみ）確認
   - 「結び（マッチング）」アクション
3. **文 (チャット一覧)**
   - マッチング済みユーザーとのチャットリスト
   - チャットルーム
4. **身上書 (マイページ)**
   - プロフィール編集
   - 設定、通知設定

## 男性ユーザー

1. **川 (ホーム)**
   - 流れてくる上の句（短冊）一覧
   - タップして詳細表示 -> 「返歌」フォーム
2. **詠み履歴**
   - 自分が送った返歌の一覧とステータス（既読/マッチ成立など）
3. **文 (チャット一覧)**
   - マッチング済みユーザーとのチャットリスト
   - チャットルーム
4. **身上書 (マイページ)**
   - プロフィール編集
   - 設定、通知設定

## 管理画面 (Admin)

- ユーザー管理（凍結/解除）
- 通報管理
- 本人確認承認フロー
