# 要件定義書 - Utagaki (歌垣)

## 1. プロジェクト概要

**プロジェクト名**: Utagaki (歌垣)

**コンセプト**:
外見重視のマッチング疲れを解消し、感性の一致（笑いのツボや価値観）を重視するマッチングアプリ。
平安時代の「歌垣」を現代風にアレンジし、和風モダンな世界観で「言葉」から始まる出会いを提供する。

**ターゲット層**:

- 言葉遊びやサブカルチャーが好きな人
- 既存の「スワイプ式」アプリに疲弊している人
- 落ち着いた関係を求めている人

## 2. ユーザーフロー（体験の流れ）

### アプリの肝：マッチング成立までの流れ

1. **【女性】上の句投稿**
   - 女性が「5・7・5（または自由律）」を入力して投稿する。
2. **【男性】ランダム受信**
   - 男性側の画面に、女性たちが投稿した「上の句」が流れてくる（川を流れる短冊のようなイメージ）。
3. **【男性】下の句詠み**
   - 気に入った上の句に対して「7・7（または自由律）」を入力して返信する。
4. **【女性】選定**
   - 女性のもとに、複数の男性からの「下の句」が届く。
   - 男性プロフィールはまだ「アイコン（家紋/イラスト）」のみ表示。
5. **【マッチング】**
   - 女性が気に入った下の句を選び、「結び（いいね）」を押す。
6. **【正体解禁・チャット開始】**
   - マッチング成立演出（御簾が上がる/扇子が開く）とともに、互いの**顔写真と詳細プロフィール**が公開される。
   - 個別のメッセージやりとり（文）が可能になる。

## 3. 機能要件リスト (MVP)

### A. ユーザー管理

- **会員登録/ログイン**:
  - LINE ログイン、Google ログイン、メールアドレス
- **プロフィール作成**:
  - ニックネーム、年齢、居住地
  - **アイコン（マッチング前）**: 家紋、妖怪、鳥獣戯画風動物、能面などのジェネレーター/選択式
  - **顔写真（マッチング後公開）**: マッチング成立後に互いに公開される写真
- **本人確認**:
  - 身分証アップロード機能（出会い系サイト規制法準拠のため）

### B. マッチング機能（ゲーム部分）

- **上の句投稿 (女)**:
  - テキスト入力フォーム（縦書き推奨）
  - 背景色・フォント選択（感情表現）
  - 制限時間設定（投稿から 24 時間で消える等）
- **広場/タイムライン (男)**:
  - 「川」の UI。流れてくる上の句（短冊/笹舟）を拾う。
  - 新着順/人気順/ランダム
- **下の句返信 (男)**:
  - 選択した上の句に対する返信フォーム
- **回答一覧 (女)**:
  - 自分の上の句についた下の句リスト
- **選定アクション (女)**:
  - マッチング成立ボタン（「結び」）

### C. コミュニケーション

- **チャットルーム（文）**:
  - テキストメッセージ、画像送信
- **通知**:
  - 女：下の句がついた時
  - 男：自分の下の句が選ばれた時

### D. 安全対策・管理機能

- **NG ワードフィルター**: 卑猥な言葉・暴言の伏せ字化/ブロック
- **通報・ブロック機能**: 不快なユーザーの遮断

## 4. ビジネス・運用要件

- **写真・プロフィール開示タイミング**:
  - マッチング成立までアイコンのみ。成立後に写真公開。
- **マネタイズ (MVP 後の検討含む)**:
  - 男性月額制（メッセージし放題「風流人」プラン）
  - アイテム課金（おひねり）：金箔の短冊など、目立たせる機能
- **コンテンツの鮮度**:
  - 投稿の 24 時間制限（ストーリー形式）で過疎感を防ぐ
